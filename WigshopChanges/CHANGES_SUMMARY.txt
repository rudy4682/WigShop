===============================================
NOTES FOR FRESH CLONE SETUP
===============================================

To apply these customizations to a fresh Dawn 15.4.0 clone:

1. Copy all files from WigshopChanges subdirectories to their corresponding
   locations in the new Dawn clone

2. Review and merge any new Dawn features that may have been added since 15.4.0

3. Test thoroughly:
   - Header layout (especially top-left logo position)
   - Search functionality (should be Dawn default)
   - Product pages with variant swatches
   - Media gallery
   - Mobile responsiveness
   - Collection pages
   - Blog/article pages

4. Ensure theme settings are configured in the Shopify admin

5. Run theme-check before deploying:
   .\.Todos\run-theme-check.ps1
